* Install

#+BEGIN_SRC sh
git clone https://gitlab.com/jadecell/dwm.git
cd dwm/
sudo make install
#+END_SRC

Add to the xinitrc file in the users home directory
#+BEGIN_SRC sh
exec dwm
#+END_SRC
